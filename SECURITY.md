# Security Policy

## Supported Versions

Security updates are provided for the latest tagged release and the `main` branch.

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues.

Use one of the following private channels:

- GitHub Security Advisories (preferred)
- Email: `security@saga.co.id`

Please include:

- A clear description of the issue
- Steps to reproduce
- Potential impact
- Suggested mitigation (if available)

## Response Process

- We will acknowledge receipt as soon as possible.
- We will investigate and validate the report.
- We will coordinate a fix and responsible disclosure timeline.
- We will credit reporters where appropriate and requested.
